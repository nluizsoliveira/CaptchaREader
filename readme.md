# T3 Algoritmos 1 - Captcha

## Especificações
### Objetivo
Ler uma imagem captcha em .pgm e retornar a sequência númerica alí descrita

###Como utilizar
1 Compilar o arquivo: 
```
gcc main.c -o main
```
2- Escolher o nome de um dos 20 captchas de ```"./captchas/..."``` 
```
captcha_20.pgm
```

### Contexto

Os arquivos **PGM (Portable GrayMap)** podem ser preto e brancos, em escala de cinza ou coloridos. A primeira dupla de caracteres define de qual maneira a imagem será renderizada. 

P1 - PBM - Preto e Branco
P2 - PGM - Escalas de Cinza
P3 - PPM - Colorido

Em seguida, são definidos **altura m** e **largura n** da imagem, o **valor máximo dos tons de cinza** e enfim o **valor de cada pixel**.


## Processamento

- São carregados captcha e todas as máscaras na memória heap. 
- O captcha é tratado em busca de diminuição de ruído/salt, modificando cédulas centrais cujos 8 vizinhos adjacentes possuem todos o mesmo valor para este mesmo valor.
- Por fim, cada uma das 10 máscaras é comparada píxel a píxel em uma área de mesmo tamanho das máscaras, vertical e depois horizontalmente. Caso a distribuição dos pixels na área seja pelo menos 90% igual à de uma máscara, é impresso o índice da máscara e a área é shiftada para que a busca re-comece.
